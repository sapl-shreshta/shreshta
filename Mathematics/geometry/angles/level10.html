✏️ Constructing Angles with Compass & Straightedge

Before protractors existed, mathematicians constructed angles using only a compass and straightedge. These techniques are still valuable today!

Constructing a 90° Angle

Steps:

  1. Draw a straight line
  2. Choose a point on the line (vertex)
  3. Using compass, draw a circle centered at the vertex
  4. Mark the two intersection points on the line
  5. From each intersection, draw larger circles (same radius)
  6. Connect the vertex to where these circles intersect (above the line)
  7. This creates a perfect 90° angle!

📐 Constructing Other Common Angles

Constructing a 60° Angle

This uses an equilateral triangle since all angles in an equilateral triangle are 60°!

  • Draw a line segment
  • Using compass, set radius to the segment length
  • Draw circles from both endpoints
  • Connect an endpoint to the circle intersection point
  • You get a 60° angle!

Constructing 45° and 30°

  • 45°: Construct 90°, then bisect it (divide it in half)
  • 30°: Construct 60°, then bisect it
  • Any angle: Use angle bisector construction

🔍 Angle Bisector Construction

An angle bisector divides any angle into two equal parts (like cutting it perfectly in half).

How to Bisect an Angle:

  1. Place compass at the angle's vertex
  2. Draw an arc that crosses both sides of the angle
  3. From the two arc intersections, draw circles with the same radius
  4. These circles intersect at a point
  5. Draw a line from the vertex through this intersection point
  6. This line perfectly bisects the angle!

🧠 Expert Problem-Solving Challenges

Challenge 1: Multi-Angle Problem

Setup: Two parallel lines crossed by a transversal. The transversal also crosses another line, creating a complex angle pattern.

Challenge: Find all 12+ angles if you only know one angle is 65°.

Skills needed: Parallel line angles, linear pairs, vertical angles, supplementary angles

Challenge 2: Triangle Construction

Setup: You know three angles of different triangles and some side lengths.

Challenge: Determine if the angles are valid, construct the triangles, and calculate missing sides using trigonometry.

Skills needed: Triangle angle sum rule, Pythagorean theorem, trigonometry

Challenge 3: Polygon Analysis

Setup: You have an irregular polygon with some angles and side lengths given.

Challenge: Find all missing angles, areas, and perimeters.

Skills needed: Polygon angle formulas, decomposition into triangles, geometry
